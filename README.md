# test-webstie
a simple test website with django

# all pip commands:
```shell
pip3 install django
pip3 install mysqlclient
```
# all apt commands:
```shell
sudo apt install libmariadb-dev
```
# all setup commands:
```shell
python3 manage.py makemigrations
python3 manage.py migrate
python3 manage.py runserver
```
